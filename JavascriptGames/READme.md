# JavaScript Games

A collection of browser-based games built with vanilla JavaScript and HTML5 Canvas.

## Games

### [Dodger](https://tylerjudson.github.io/Portfolio/JavascriptGames/Dodger/)
A space shooter game where you pilot a spaceship and try to survive as long as possible.
- Use arrow keys to navigate
- Press F to fire projectiles
- Press spacebar to clear the screen (limited uses)
- Collect triangles for extra lives
- Collect stars for additional screen clears

### [Star Catchers](https://tylerjudson.github.io/Portfolio/JavascriptGames/StarCatchers/)
A two-player competitive game where players race to collect stars.
- Red player: WASD controls
- Blue player: Arrow keys
- Yellow stars: +1 point
- Same-color stars: +3 points
- Opposite-color stars: -1 point
- Highest score at the end wins

### [Whiteboard](https://tylerjudson.github.io/Portfolio/JavascriptGames/Whiteboard/)
A digital drawing application with full customization options.
- Multiple brush colors
- Adjustable line widths
- Flat and round brush shapes
- Eraser mode with separate settings
- Clear screen functionality

### [Base Game](https://tylerjudson.github.io/Portfolio/JavascriptGames/BaseGame/)
A template for canvas-based games with basic features like score tracking and start/restart UI.

### [Base Platformer](https://tylerjudson.github.io/Portfolio/JavascriptGames/BasePlatformer/)
A platformer game template with keyboard and mobile touch controls for movement and jumping.

### [Canvas Resize](https://tylerjudson.github.io/Portfolio/JavascriptGames/CanvasResize/)
A utility project for handling responsive canvas sizing across different screen sizes.

### [Star Shower](https://tylerjudson.github.io/Portfolio/JavascriptGames/StarShower/)
A visual animation project featuring star effects on canvas.
